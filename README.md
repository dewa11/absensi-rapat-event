Aplikasi Absensi rapat & event tanpa perlu TTD kertas untuk kantoran & Rumah Sakit/Klinik berbasis Web dengan menggunakan framework Flight
Mohon atur setting koneksi db nya terlebih dahulu
silahkan ganti nama RS dengan perusahaan atau RS/Klinik masing2
ganti logo perusahaan juga silahkan
Menu Buat Absen untuk membuat rapat atau event
menu data absen untuk melihat meeting yg telah dibuat
di dalam nya ada feature generate link & QR code
user tamu hanya perlu scan atau klik link yg telah dibagikan oleh admin
kemudian mengisi form yg telah diberikan layaknya gform
silahkan seed db user pass admin nya sesuai keinginan
